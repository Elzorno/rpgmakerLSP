# Journey II — The Forgotten World

Journey II begins when Kai leaves Home Island and discovers that every region lives among old systems they do not understand.

---

## Purpose

Journey II expands the game from a local mystery into a world-spanning pattern.

The first region in this journey is Coalmouth.

---

## Core Question

> What happened to the world?

Journey II should not answer this all at once. Each region should reveal one part of the hidden truth.
