# Coalmouth NPC Roster

This roster defines the first-pass NPC set for Coalmouth.

Vera is currently the major character object. The remaining NPCs are role-based placeholders until named and promoted.

---

## Purpose

Coalmouth NPCs should make the town feel like a working community under stress.

Each NPC should reinforce the industrial-control theme without using modern technical vocabulary.

---

| ID | Name | Role |
|---|---|---|

Coalmouth NPCs should support at least:

```text

Mine Lockdown
After Vera Joins/Investigates
After Node Six Stabilized
```

---

## Promotion Rule

Promote a placeholder NPC into a full object if they:

- start a quest,
- recur after Coalmouth,
- need unique portrait/sprite work,
- carry emotional stakes,
- become important to Vera’s arc.
