# Home Island Location Art Prompt Packet

This packet defines first-pass visual direction for Home Island map references and RPG Maker MZ asset production.

## Global Direction

Home Island should feel warm, hopeful, rural, and quietly mysterious. Old technology should appear as salvage, strange stonework, reused metal, or sacred ruins rather than clean science fiction.

## Ashford

Create an RPG Maker MZ-compatible pixel-art village reference for a warm starting village built around the shell of an old factory. Include timber homes, gardens, warm-stone vents, old metal panels reused as fences, flowers near cracked concrete, and golden morning light.

## Rustshore Docks

Create an RPG Maker MZ-compatible pixel-art harbor reference for a small island dock with fishing boats, rope, sea grass, patched wooden piers, a dockmaster hut, and a small lighthouse built partly from ancient metal.

## Fogfen Marsh

Create an RPG Maker MZ-compatible pixel-art marsh reference with reeds, shallow water, bent trees, fog, mossy stones, and faint blue-white glows beneath the water. The mood should be mysterious but not horror-focused.

## Glassfield Ruins

Create an RPG Maker MZ-compatible pixel-art ruin reference with cracked glass-like panels, grass, flowers, broken concrete, half-buried metal ribs, and faint inactive lights. The mood should be beautiful and archaeological.

## Skyreach Hill

Create an RPG Maker MZ-compatible pixel-art hill reference with wind, tall grass, carved geometric stones, open sky, and a cave entrance near the top. The old technology should read as sacred ancient carving.

## Hidden Cave

Create an RPG Maker MZ-compatible pixel-art cave reference with reflective stone walls, carved panels, dim blue-white light, trial chambers, and an inner sanctum pedestal for the Sword.

## Sealed Node

Create an RPG Maker MZ-compatible pixel-art dungeon reference where cave and ruin details transition into old infrastructure. Include stone corridors, glass panels, inactive machinery, warning lights, and a blue-white relay core chamber.

## Acceptance Criteria

- Each prompt can guide RPG Maker MZ map or tileset work.
- Home Island remains hopeful.
- Technology is subtle except in the Sealed Node.
- Generated concepts remain original and usable as references.
