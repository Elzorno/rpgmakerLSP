# Home Island Battler Prompt Packet

This prompt packet defines initial image-generation prompts for Home Island enemy battlers.

All assets should be front-view RPG Maker MZ-compatible battlers with transparent backgrounds unless stated otherwise.

---

## Global Style Requirements

Use these requirements for every prompt:

```text
front-view RPG Maker MZ battler, transparent background, readable silhouette, charming classic JRPG-inspired style, original design, no text, no UI, no background, clean edges, game asset ready
```

Avoid direct imitation of Dragon Quest monsters.

---

## MON-GEL-001 — Meadow Gel

```text
Create a front-view RPG Maker MZ battler of an original early-game gel monster called Meadow Gel. It has a round teardrop-like semi-transparent body, simple expressive face, soft green-blue palette, tiny grass flecks inside the gel, charming and harmless first-enemy personality, readable silhouette, transparent background, no text, no UI, no background.
```

---

## MON-GEL-002 — Marsh Gel

```text
Create a front-view RPG Maker MZ battler of an original early-game gel monster called Marsh Gel. Use the same general silhouette family as Meadow Gel but with a darker swamp-green palette, muddy base, tiny reed fragments, faint pale violet-blue glow inside the gel, slightly uneasy but still charming, transparent background, no text, no UI, no background.
```

---

## MON-RAT-001 — Ash Rat

```text
Create a front-view RPG Maker MZ battler of an original early-game monster called Ash Rat. It is a small scruffy gray-brown island rat with oversized ears, alert eyes, curved tail, warm ash-colored fur, charming classic JRPG monster style, not realistic horror, readable silhouette, transparent background, no text, no UI, no background.
```

---

## BOS-N07-001 — Node Seven Guardian

```text
Create a front-view RPG Maker MZ boss battler of an ancient stone-and-metal guardian called Node Seven Guardian. It is a degraded security construct with a cave-ruin body, old metal ribs, stone armor plates, a blue-white glowing core, small unstable red cracks, sacred guardian silhouette, early-game boss scale, mysterious but not too horrific, transparent background, no text, no UI, no background.
```

---

## Variant Production Rule

Generate the Meadow Gel first. Use it as the family reference for Marsh Gel so variants feel related.

Generate Ash Rat separately.

Generate Node Seven Guardian at larger boss scale.

---

## Acceptance Criteria

- Each image has transparent background.
- Each battler is readable at RPG Maker scale.
- Gel variants clearly share a family silhouette.
- Designs feel original and do not copy protected monster designs.
- Node Seven Guardian feels like a guardian system, not a demon.
