# Atlas Roadmap

Atlas is now the primary deliverable for _The Last Sword Protocol_.

RPG Maker MZ is the first implementation target. Atlas is engine independent.

---

## Atlas v1.0 Goal

Atlas v1.0 is complete when another contributor could begin implementation without inventing foundational world rules, story structure, gameplay meaning, production standards, or hidden technology logic.

Atlas v1.0 does not need every line of dialogue or every stat table, but it must define the project well enough that future work can proceed coherently.

## Milestone 2 — World

Purpose: Define the world as a playable, explorable, internally coherent place.

Deliverables:

- World timeline
- Four Ages expanded
- Overworld geography
- Region packets
- Location packets
- Transportation model
- Cultures and economies
- Forgotten technology reference

Completion rule: Every required location has a purpose, hidden reality, story role, and implementation packet.

## Milestone 3 — Story
