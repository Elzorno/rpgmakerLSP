# 00 Foundation

The Foundation section defines how Atlas itself works.

Before creating or changing world, story, gameplay, art, technical, or implementation pages, contributors should understand the Foundation documents.

---

## Foundation Documents

| Document | Purpose |
|---|---|
| Atlas Operating System | The workflow every contributor and AI agent follows |
| Atlas Roadmap | Milestones for Atlas v1.0 |
| Atlas ID Specification | Permanent ID system for pages and concepts |
| Metadata Schema | Required front matter and production metadata |
| Page Template | Standard structure for future Atlas pages |

---

## Foundation Principle

Atlas should be:

- engine independent,
- searchable,
- cross-referenced,
- AI friendly,
- human friendly,
- version controlled,
- deterministic,
- expandable,
- self-documenting.

Atlas is not a folder of notes.

Atlas is the source of truth.
