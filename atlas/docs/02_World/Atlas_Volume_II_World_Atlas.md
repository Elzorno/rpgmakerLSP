# Atlas Volume II — World Atlas

## Purpose

Atlas Volume II is the canonical production document for building The Last Sword Protocol one region at a time.

Unlike the Creative Bible, this volume is an executable specification. Nothing should remain to be invented during implementation.

## Design Philosophy

Each region is completed vertically before moving to the next.

For every region Atlas must fully define:

1. World maps
2. Every NPC
3. Every dialogue line
4. Every quest
5. Every cutscene
6. Every switch and variable
7. Every treasure
8. Every hidden item
9. Every enemy encounter
10. Every boss
11. Every audio cue
12. Every art asset prompt
13. Every implementation packet
14. Every Codex handoff
15. Playtest checklist and acceptance criteria

## Production Order

- Phase 1 — Home Island (Vertical Slice)
- Phase 2 — Coalmouth
- Phase 3 — Athenaeum
- Phase 4 — Frontier Regions
- Phase 5 — Final Network

## Completion Definition

A region is complete only when Codex can implement it without inventing mechanics, assets, dialogue, or event logic.

Atlas is the design authority. AI agents are builders.
