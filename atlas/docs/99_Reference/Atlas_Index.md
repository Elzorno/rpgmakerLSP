The Atlas Index is the beginning of the searchable knowledge graph for _The Last Sword Protocol_.

It provides a centralized reference point for major people, places, systems, concepts, documents, and production terms.

This document answers:

As Atlas grows, this page should become the first stop for both humans and AI agents.

## A

### Archive

Recovered old-world memory and system data accessed through the Sword and relays.

Related: Archive Recovery, Memory Shrine, Project Excalibur, Relay Node.

### Archive Recovery

Story variable representing how much of the Sword / Project Excalibur archive has been restored.

Related: Story Structure Bible, RPG Maker MZ Bible.

### Ashford

Kai's home village on the starting island, built around the shell of an old-world factory.

Related: Home Island, Grandmother Elara, Skyreach Hill.

### Athenaeum

Library-city built in and around a collapsed university/archive complex.

Related: Eldon, Signal Tower, Integrity.

### Authentication

Hidden cybersecurity principle behind blessings, Sword recognition, sealed doors, and shrine access.

Related: Atlas Concordance, Cybersecurity Layer Bible, Project Excalibur.

## C

### Chain of Trust

Hidden cybersecurity model behind divine chains, relay validation, Sword authority, and final revocation.

### Cipher
