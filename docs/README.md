# The Last Sword Protocol — Project Atlas

**Project Atlas** is the canonical design and production knowledge base for *The Last Sword Protocol*.

Atlas is not a loose notes folder. It is the project authority. Every future map, monster, item, quest, cutscene, asset prompt, RPG Maker event, and AI-generated implementation should be measured against Atlas before it is accepted into the game.

> If implementation and Atlas disagree, Atlas wins.

---

## Current Status

Implementation is paused. The current RPG Maker MZ project is treated as a capability test and prototype. Existing assets, maps, database entries, and event work may be reused, revised, or discarded as needed.

The current phase is **pre-production planning**.

The goal of this phase is to create a near-final design package so implementation can restart later with clear direction.

## Canonical Hierarchy

When documents disagree, use this order of authority:

1. **Project Constitution** — immutable project identity.
2. **Studio Manifesto** — creative philosophy.
3. **Creative Bible** — master design authority.
4. **Player Promise** — player-facing experience standards.
5. **Development Standards** — workflow and implementation rules.
6. Future specialized bibles: world, story, character, art, technical, and bestiary.

---

## Document Lifecycle

Each document should move through these states:

```text
Draft → Review → Approved → Locked
```

A locked document is canonical. If implementation later reveals a necessary change, update the relevant Atlas document first, then update the game.

---

## Design North Star

*The Last Sword Protocol* should feel like a classic Dragon Quest-inspired JRPG first.

The cybersecurity, AI, post-apocalyptic history, and hidden engineering layer are not gimmicks. They are the secret structure beneath the fantasy.

The player should begin with wonder, gradually discover patterns, and eventually realize that the magic of the world was forgotten technology all along.

The game is not about technology replacing humanity.

It is about people choosing wisdom, responsibility, curiosity, and hope.
