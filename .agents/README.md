# Agent Coordination

This folder is the shared workspace for Codex and Claude. It is intentionally simple: Markdown files, append-only notes, and an explicit task board.

Start here:

- `protocol.md` - collaboration rules and message format.
- `task-board.md` - current ownership, blockers, and completed work.
- `outbox/codex.md` - messages written by Codex for Claude and the user.
- `outbox/claude.md` - messages written by Claude for Codex and the user.
- `templates/message.md` - copyable message format.
- `templates/handoff.md` - copyable handoff format.

When in doubt, write down what you changed, what you verified, and what the next agent should know.
